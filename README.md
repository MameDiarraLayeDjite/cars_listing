# Mini-application de Vente de Voitures

## Description

Cette mini-application web permet aux utilisateurs de consulter une liste de voitures disponibles à la vente et de voir les détails de chaque véhicule. Elle inclut aussi une interface d'administration sécurisée permettant de gérer le catalogue de voitures (CRUD).

---

## Fonctionnalités principales

### Interface publique

- Liste paginée de toutes les voitures disponibles avec image, nom complet, kilométrage, lieu, prix de vente.
- Page détail d’un véhicule avec informations complètes et galerie de photos.

### Interface d’administration (Dashboard)

- Authentification sécurisée (login / mot de passe).
- Consultation, recherche et filtrage des voitures.
- Ajout, modification, suppression de voitures avec validation.
- Confirmation avant suppression.

---

## Prérequis

- Node.js v18+ (https://nodejs.org/)
- MySQL 8+ (https://dev.mysql.com/)
- npm (inclus avec Node.js)
